# Credits

SlickSync is a fork of [Syncio](https://github.com/iamneur0/syncio) by
**neur0**, with additional features and ideas credited to
[AIOManager](https://github.com/Sonicx161/AIOManager) by **Sonicx161**.

## What SlickSync adds on top of Syncio

- **Live "Now Playing" tracking**, sourced in part from an AIOStreams proxy
  integration that detects active streams in real time, independent of
  library-poll delays.
- **AIOMetadata-powered poster enrichment** for streams that have no
  library metadata match.
- Various theming, badge, and dashboard changes specific to this fork.
